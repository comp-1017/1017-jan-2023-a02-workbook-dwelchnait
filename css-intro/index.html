StyleSheet Similar to embedded however, the selectors and rules are code in a separater physical file. This file can be shared throughout the entire web site. Thus, all css is in one common place. This is the preferred technique to use. To use the style sheet on a particular web page, you must reference the stylesheet using a <link> tag.
