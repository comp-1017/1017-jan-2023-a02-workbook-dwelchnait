Creating Site Navigational Links

Navigational links are simply anchor tags with appropriate relative addressing for the href parameter

  • page to page within the same folder
  • page to page between folders
  • stay on the current page using the Pound symbol (#)
  • stay on the current page and jump to another location

Until otherwise told, your nav semantic tag my have an optional h2 heading. Yes, you will get a document outline error BUT it will be acceptable. Until we cover formatting of navigational menus, use a simply unordered list.
