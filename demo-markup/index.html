Open Bridge from Photoshop

To open Bridge from within Photoshop, go up to the File menu (in Photoshop) in the Menu Bar along the top of the screen and choose Browse in Bridge. If you are the type who likes keyboard shortcuts, you can choose the same command by pressing Ctrl+Alt+O (Win) or Command+Option+O (Mac).

  1. Go to the file menu
  2. From the drop down select Browse in Bridge
  3. Navigate to the file you wish to open
  4. From the open file dialogue box click on the open button
